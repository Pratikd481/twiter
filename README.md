

## Installation 

- Git clone https://github.com/Pratikd481/twiter.git
- Put env file in the root folder
- Setup db and update env as per the db setup
- composer install
- npm install
- npm run prod
- php artisan migrate --seed
- php artisan serve
- open http://localhost:8000 in browser
